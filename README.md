# XMLChecker
Cyber Ninjas XML Checker

## Building Docker Image

```sh
$ docker build . -t cyberninjas/xmlchecker
```

## Using From Command Line
* Required Python v3

Grab the dependencies.
```sh
$ pip install defusedxml
```

Run the script.
```sh
$ XMLChecker.py <xmlfile.xml>
```
